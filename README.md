# Benchmark-test

## Benchmark for simple function
- cpp
- cpp_mt
- simd
- simd_mt
- simd_fma
- simd_fma_mt
- python
- python_mt
- python_numpy
- python_numpy_mt

This benchmark tests the performance of a simple function beetween 2 languages: C++ and Python.
It compares the performance of single-threaded and multi-threaded implementations, as well as SIMD and FMA optimizations in C++.

## Usage
- run `run_benchmark_avg.py` for testing
- run `plot_avg.py` to see comparison chart

test files are located in `out/` directory
